This site is hand-built and deliberately small: plain HTML and CSS,
assembled by [Jekyll](https://jekyllrb.com) and served by GitHub Pages.
No analytics, no trackers, no JavaScript beyond a light switch.

The type is [Newsreader](https://fonts.google.com/specimen/Newsreader),
an editorial serif drawn for on-screen reading. The hairline border is
a frame — the slow kind.

Content lives in plain-text files in a
[public repository](https://github.com/scottallisonsi/scottallisonsi.github.io);
essays are written in Markdown, everything else in small YAML lists.
